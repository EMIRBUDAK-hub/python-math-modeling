# Modélisation mathématique en Python

Projets académiques réalisés en Python dans le cadre d’un cycle préparatoire intégré
d’école d’ingénieur, axés sur la modélisation mathématique, l’intégration numérique
et l’analyse de fonctions.

## Contenu du dépôt

### Intégrales multiples et modélisation
- Intégrales doubles et triples sur différents domaines (rectangle, triangle, disque, sphère)
- Changement de variables et coordonnées polaires / sphériques
- Comparaison entre méthodes numériques et résultats analytiques
- Applications physiques (flux thermique)
- Visualisations et grilles de calcul

### Analyse de fonctions
- Étude de fonctions de deux variables
- Visualisations 2D et 3D
- Courbes de niveau
- Champs de gradient et recherche de points critiques

## Technologies utilisées
- Python
- NumPy
- SciPy
- Matplotlib
